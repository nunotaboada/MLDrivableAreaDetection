class model.DoubleConv(in_channels, out_channels, dropout_rate=0.1)[source]

Bases: Module

A double convolution block for the U-Net architecture.

Applies two consecutive convolutional layers, each followed by batch normalization, ReLU activation, and a dropout layer to prevent overfitting.

Parameters:

  • in_channels (int) – Number of input channels.

  • out_channels (int) – Number of output channels.

  • dropout_rate (float, optional) – Dropout probability to prevent overfitting. Defaults to 0.1.

class model.UNET(in_channels=3, out_channels=3, features=[32, 64, 128, 256])[source]

Bases: Module

U-Net architecture for semantic segmentation.

Implements a U-Net with an encoder-decoder structure, including skip connections, for multiclass segmentation tasks. The model consists of a downsampling path (encoder), a bottleneck, and an upsampling path (decoder), with a final convolution to produce the output segmentation map for three classes (background, driveable, lanes).

Parameters:

  • in_channels (int, optional) – Number of input channels (e.g., 3 for RGB images). Defaults to 3.

  • out_channels (int, optional) – Number of output channels/classes (e.g., 3 for multiclass segmentation). Defaults to 3.

  • features (list, optional) – List of feature sizes for each level in the encoder and decoder. Defaults to [32, 64, 128, 256].

initialize_weights()[source]

Initializes weights for Conv2d, ConvTranspose2d, and BatchNorm2d layers.

Uses Kaiming uniform initialization for convolutional layers and constant initialization for batch normalization layers to improve training stability.

Notes

  • Conv2d and ConvTranspose2d weights are initialized with Kaiming uniform (ReLU nonlinearity).

  • BatchNorm2d weights are set to 1, and biases are set to 0.
